1.ชื่อโปรเจกต์
TaskFlow Planner

2.คำอธิบายโปรเจกต์
ระบบจัดการงานและโปรเจกต์ ผู้ใช้สามารถสร้างโปรเจกต์ เพิ่มงาน แก้ไข ลบ และติดตามความคืบหน้าได้ มี Dashboard แสดงสถิติและกราฟภาพรวม มีระบบรายงานวิเคราะห์ประสิทธิภาพการทำงาน

3.เทคโนโลยีที่ใช้
    -Express
    -EJS
    -Sequelize
    -SQLite
    -bcrypt
    -express-session
    -Chart.js

4.วิธีติดตั้ง
    4.1 Clone โปรเจกต์จาก GitHub
    4.2 เข้าโฟลเดอร์โปรเจกต์
    4.3 ติดตั้ง dependencies ด้วยคำสั่ง

5.วิธีรันโปรแกรม
npm install
npm run seed
npm run dev

จากนั้นเปิดใช้งานที่
http://localhost:3000

6.รายการฟีเจอร์

    -ระบบ Login / Logout
    -แยกสิทธิ์ Admin และ User
    -CRUD Users (เฉพาะ Admin)
    -CRUD Projects
    -CRUD Tasks
    -Task Logs
    -Dashboard แสดงสถิติ
    -Reports
